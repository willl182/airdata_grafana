# Technical Finding / Milestone

**Context**: Grafana downloader V2 TypeScript migration and plan synchronization.

## The Discovery

The local Grafana engine can be migrated incrementally to TypeScript while preserving the existing CommonJS package mode and public CLI commands. The practical compatibility pattern is:

- keep `scripts/*.js` as wrappers;
- compile `src/grafana/*.ts` and scripts to `dist/`;
- run public commands through `pnpm run build` before invoking `dist/scripts/*.js`;
- keep `dist/` out of git.

This allowed the project to gain shared types and typechecking without changing the user-facing command surface.
